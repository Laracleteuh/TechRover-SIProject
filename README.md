Welcome to the Tech Rover adventure!
This exciting project invites us to design, model, simulate, and build a miniature all-terrain vehicle measuring approximately 30 cm, capable of moving autonomously on different types of surfaces.

This project is being carried out as part of our engineering studies. 

In order to help future engineers design their own projects and better understand the technologies involved, we will make our code files, electrical diagrams, and 3D drawings of the parts we have created available to them.

Updates to the code will be posted regularly online to meet the project's requirements, and update notes will be provided. Updates to the part diagrams will also be posted online so that any problems detected can be corrected, allowing you to carry out your study/project in the best possible conditions.

The repository will be organized into folders according to the type of file it contains: 

> Code

> Schematics

> Parts

> Latest official update
